
  # ASCII Generator

  This is a code bundle for Website Page Layout Coding. The original project is available at https://www.figma.com/design/UJUPs8W0Hm9msdDMtgpefE/Website-Page-Layout-Coding.

  ## Running the code
  ## EN
  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## RU

  Простой проект для генерации ASCII-арт изображений и просмотра галереи.
  
  ## Установка
  
  1. Установить зависимости:
     ```bash
     npm install
     ```
  
  2. Запустить проект:
     ```bash
     npm run dev
     ```
  
  ## Что есть в проекте
  
  - Главная страница
  - Конвертер изображений в ASCII
  - Галерея сохранённых результатов
  
  ## Примечание
  
  Файлы `node_modules` в репозиторий не добавляются.
